一、String类

  • 声明：

    String是字符串，一连串字符的序列，他是一个特殊的数据类型
String类实例化有两种方式
String a = "abc";
String a1 = "abc";
String a2 = new String("abc");
以前在学基本数据类型的时候我们判断两个数值是否相等用的是  == 号，现在咱学到字符
串，他们之间值是否相等比较的话用.equals(); 那 == 用在两个字符串比较的时候，他比较的是内存地址，了解即可

  • 两种方式区别：

      第一种：对象值存在常量池中只开辟一块内存空间,字符串内容可以自动入池,以供下次使用.
a1和a2指向同一块内存地址 因为String用的最多，很特殊，因为对于String来讲使用直接赋值的
方式会在字符串池中保存内容。如果之后在声明字符串的时候发现内容一样，则不会重新开辟空间，
而是从内存池中取出数据继续使用。
  第二种：这里创建了2个实例化对象,一个是String的匿名对象"Hello",另一个是使用关键字new 
实例化的String类对象.开辟两块堆内存空间,有一块将成为垃圾,并且不能自动入池,
需要使用intern()手工入池.

  • 使用

    1、比较字符串方法 

    boolean equals :将该字符串与指定对象比较,如果二者包含的字符序列相等,则返回true,否则返回false.
eg:
String str1 = "iPhone";
String str2 = "Android";
if (str1.equals(str2)) 
{System.out.println("相同");
} else 
{System.out.println("不相同");
}

    2、查找方法 

      int indexOf(int c):找出c字符在该字符串第一次出现的位置. 
      int indexOf(int c, int start):找出c字符在该字符串中从start开始后第一次出现的位置.  
      int indexOf(String str):找出str字符串在该字符串中第一次出现的位置. 
      int indexOf(String str , int start): 找出str字符串在该字符串中从start开始后第一次出现的位置.  
      计算长度:int length(): 返回当前字符串长度.
      eg:
        String outStr = "I want to have an iPhone";
        int index1 = outStr.indexOf('a');
        int index2 = outStr.indexOf('a',4);
    int index3 = outStr.indexOf("iPhone");

    3、替换方法:

    String replace(char oldChar , char newChar ):将字符串中的第一个oldChar替换成newChar.
      判断开始或结束
        boolean startWith(String prefix):该String对象是否以prefix开始.
     boolean startWith(String prefix ,int toffset): 
    该String对象从toffset位置算起,是否以prefix开始. 
    boolean endWith(String prefix):该String对象是否以prefix结束.
        eg.
        String outStr = "I want to have an iPhone";
        str1 = str1.replace('P', 's');
        str1 = str1.replaceAll("iP", "hhh");

        // 判断是否已某某结尾
        String str = "icon.jpg";
        if (str.endsWith(".jpg")) {
        str = str.replace(".jpg", ".png");
        System.out.println(str);
        }
        //判读是否已某某开始
        String number = "13188866666";
        if (number.startsWith("131")) {
            int a = number.length();
            System.out.println(a);
        }

  String trim(): 去除字符串中的空格.
  eg.
 // 去空格 首尾

###二、StringBuffer类
使用 StringBuffer sb = new StringBuffer();
StringBuffer对象代表一个字符可变的字符串,当一个StringBuffer被创建以后,
通过StringBuffer提供的append(),insert(),substring()等方法可以改变这个字符串对象的字符序列.
拼接方法:
sb.append();
eg.
// 拼接字符串
StringBuffer buffer = new StringBuffer();
buffer.append(“Lanou”);
buffer.append(“Dalian”);

截取方法：
sb.subString();
eg.
// 截取
String str = buffer.substring(5);// start
System.out.println(str);String str2 = buffer.substring(2, 4);// 左闭右开，不包含右侧
插入方法:
sb.insert();
buffer.insert(5, "Me");
删除方法:
sb.delete();
eg.
buffer.delete(2, 5);// 左闭右开

三、包装类

Java语言是一个面向对象的语言，但是Java中的基本数据类型却是不面向对象的，这在实际使用时存在
很多的不便，为了解决这个不足，在设计类时为每个基本数据类型设计了一个对应的类进行代表，
这样八个和基本数据类型对应的类统称为包装类(Wrapper Class)，有些地方也翻译为外覆类或数据类型类


包装类的作用：
提供：字符串、基本类型数据、对象之间互相转化的方式
包含每种基本数据类型的相关属性，如最大值、最小值等。
简单转化  
为数字的字符串 <----> 数字        .
String 类 <----->  基本数据类型
一个是类 一个是数据类型  不在一层次上
这时候引出包装类  基本数据类型对应的包装类
eg.
String s = "123";
int i = Integer.parseInt(s);
反过来
int i = 123;
Integer in = Integer.valueOf(i);或者 Integer in  = new Integer(i);
String s = in.toString();

###五、Math

Math类定义了多个实现标准数学函数功能的基本方法 如三角函数，开根号等运算
Math类中所有的成员方法都是静态的，所以直接类名.方直接调用
常用的一些方法：
Math.round();四舍五入
Math.max();取两个数的最大值
Math.min();取两个数的最小值
Math.abs()；去绝对值
三角函数：
sin(double a)  cos()等 a表示弧度，角度与弧度计算公式：2*PI/360 *角度
pow(double a,double b):a的B次幂

随机数：
Math.random();一个[0.0~1.0]之间的数
